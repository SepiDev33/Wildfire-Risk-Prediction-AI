# Wildfire Risk Prediction Using Machine Learning

## Description

Wildfires are a major environmental and economic concern in many parts of the world. Being able to predict wildfire risk using environmental and weather conditions can help researchers and authorities better understand when fires are more likely to occur. The goal of this project is to develop a machine learning model that predicts wildfire risk based on environmental and weather data.

The project uses the **Algerian Forest Fires dataset** from the UCI Machine Learning Repository. This dataset contains environmental and fire-related features such as temperature, humidity, wind speed, rainfall, and fire weather indices. These features will be used as input to train predictive models that estimate whether wildfire conditions correspond to fire or not fire.

Several machine learning models will be implemented and compared, such as Logistic Regression, Decision Trees, and Random Forest. The dataset will first be cleaned and prepared using data preprocessing techniques such as handling missing values, normalizing data, and selecting useful features. After training the models, their performance will be evaluated using standard metrics such as accuracy, precision, recall, and F1-score.

The goal of the project is not only to build a working prediction model but also to analyze how different environmental factors influence wildfire risk. By comparing different machine learning approaches, the project will explore which models perform best and why certain variables are important predictors of wildfire conditions.

This project demonstrates how artificial intelligence techniques, specifically pattern recognition and predictive modeling, can be applied to real-world environmental problems.
